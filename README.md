# AIWD_LAB2
Artificial Intelligence in Web Development Laboratory 2

Description

In this experiment, the HTML5-based Online Event Registration & Promotional Portal created in Experiment 1 is enhanced using CSS and advanced CSS3 techniques. Styling is applied to improve the layout, typography, colors, and spacing of the webpage. Advanced CSS3 features such as Flexbox are used for layout alignment, CSS Transitions are applied to buttons and form fields for interactive effects, and Media Queries are implemented to ensure responsiveness on different screen sizes. The styling is added using an external CSS file without modifying the HTML structure.
